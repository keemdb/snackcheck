# Snackcheck! 로드맵

팀 간식 구입을 위한 체크리스트 모바일 웹앱

---

## 프로젝트 개요

| 항목 | 내용 |
|------|------|
| 프로젝트명 | Snackcheck! |
| 플랫폼 | 모바일 웹뷰 (Mobile WebView) |
| 목적 | 팀 간식 구매 장바구니 관리 및 층별 분배 |
| 회사 주소 | 서울시 서초구 서초동 1327-15 7층 7호 |

---

## 화면 구성 (3탭)

```
[장바구니] → [영수증 애니메이션] → [간식 분배]
```

---

## Phase 1 — 프로젝트 초기 세팅

- [ ] 프로젝트 디렉터리 구조 잡기 (React 또는 Vanilla HTML/CSS/JS 선택)
- [ ] 모바일 뷰포트 메타태그 설정 (`width=device-width, initial-scale=1`)
- [ ] 전역 스타일 정의 (폰트, 컬러 팔레트, 버튼 스타일)
- [ ] 탭 네비게이션 컴포넌트 (장바구니 / 간식분배)

---

## Phase 2 — 장바구니 탭

### 입력 폼
- [ ] 입력 필드 4개
  - 물건명 (텍스트)
  - 개수 (숫자)
  - 수량 단위 (예: 개, 박스, 봉 등)
  - 가격 (숫자, 원 단위)
- [ ] "추가" 버튼

### 장바구니 리스트
- [ ] 항목 추가 시 리스트에 즉시 반영
- [ ] 각 항목에 삭제 버튼
- [ ] 리스트 항목 표시 형식: `물건명 | 개수 단위 | 가격`
- [ ] 가격 합계 실시간 표시 (하단 고정)
- [ ] 항목이 없을 때 빈 상태 안내 문구 표시

### 구입완료 버튼
- [ ] 리스트에 항목이 1개 이상일 때만 활성화
- [ ] 클릭 시 영수증 애니메이션 화면으로 전환

---

## Phase 3 — 영수증 애니메이션

### 애니메이션
- [ ] 영수증이 아래에서 위로 프린트되듯 슬라이드 업 또는 롤 업 애니메이션
- [ ] 인쇄기 소리 효과 (선택)

### 영수증 내용
- [ ] 상단: "Snackcheck! 영수증" 제목
- [ ] 구입 일시 (현재 날짜/시간 자동 삽입)
- [ ] 구분선 (점선)
- [ ] 구입 목록 (물건명, 개수 단위, 가격)
- [ ] 구분선 (점선)
- [ ] 합계 금액 (굵게 강조)
- [ ] 하단: 회사 주소 `서울시 서초구 서초동 1327-15 7층 7호`

### 영수증 확인 버튼
- [ ] 영수증 하단에 "확인" 버튼 배치
- [ ] 클릭 시 간식 분배 탭으로 이동

---

## Phase 4 — 간식 분배 탭

### 분배 기준 (고정값)
| 층 | 인원 |
|----|------|
| 7층 | 10명 |
| 8층 | 6명 |

### 분배 계산 로직
- [ ] 각 물건 개수를 전체 인원(16명) 비율로 나눠 층별 분배 수량 계산
  - 7층 비율: 10/16 (≈ 62.5%)
  - 8층 비율: 6/16 (≈ 37.5%)
  - 나머지(소수점)는 반올림 처리 후 합계 검증
- [ ] 개수가 인원보다 적을 경우 처리 표시 (예: "부족 — 추가 구매 필요")

### 분배 결과 리스트
- [ ] 물건별로 7층 / 8층 수량을 나란히 표시
- [ ] 예시:
  ```
  포카칩   7층: 4개  |  8층: 2개
  음료수   7층: 7캔  |  8층: 5캔
  ```
- [ ] 다시 구매하기 버튼 → 장바구니 초기화 후 장바구니 탭으로 이동

## Phase 5 — UI/UX 다듬기

- [ ] 모바일 터치 피드백 (버튼 탭 시 시각적 반응)
- [ ] 입력 폼 유효성 검사 (빈 값, 0 이하 숫자 방지)
- [ ] 숫자 입력 시 천 단위 콤마 포맷 (예: 12,000원)
- [ ] 영수증 롱프레스 또는 공유 버튼으로 이미지 저장 (선택)
- [ ] 전체 반응형 레이아웃 (최대 너비 430px 기준)
